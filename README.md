## 📝 Giới Thiệu Dự Án

**WarehouseFoodManagement** là giải pháp phần mềm quản lý kho lương thực, thực phẩm toàn diện được phát triển bằng ngôn ngữ **Java**. Dự án được thiết kế nhằm tối ưu hóa quy trình lưu trữ, theo dõi hạn sử dụng, quản lý nhập/xuất kho và giảm thiểu tối đa tình trạng lãng phí thực phẩm cho các doanh nghiệp hoặc đại lý phân phối.

### ✨ Tính Năng Nổi Bật
*   📦 **Quản lý sản phẩm thông minh:** Phân loại thực phẩm theo danh mục (đồ tươi sống, đồ khô, đông lạnh...).
*   ⏳ **Cảnh báo hạn sử dụng:** Tự động phát hiện và cảnh báo các lô hàng sắp hết hạn.
*   📊 **Báo cáo xuất/nhập tồn:** Thống kê chi tiết lượng hàng hóa lưu kho theo thời gian thực.
*   ⚡ **Tối ưu hiệu suất:** Cấu trúc dữ liệu chuẩn giúp xử lý tìm kiếm và lọc dữ liệu cực nhanh.



## 🛠️ Công Nghệ Sử Dụng

*   **Ngôn ngữ chính:** Java (JDK 11+)
*   **Thư viện hỗ trợ:** Được đóng gói trong thư mục `lib/`
*   **Mô hình kiến trúc:** MVC (Model-View-Controller) giúp code sạch và dễ bảo trì.
